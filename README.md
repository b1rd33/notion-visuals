# notion-visuals

Self-contained HTML visuals for embedding in Notion pages via GitHub Pages.

## Structure

```
daytwo/          # DayTwo HBS case overview
artiso/          # ARTISO pitch deck visuals
...              # Future project folders
```

## Usage

Embed in Notion: `/embed` → paste `https://b1rd33.github.io/notion-visuals/<path>/index.html`
